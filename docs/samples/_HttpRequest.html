サンプルアプリ _HttpRequest

サーバーとの通信デモです。

動作確認するには、PHP（5.2.0以上）の動作するサーバーに「htdocs」フォルダ内一式をアップロードしてください。
なお、「test1.php」「test2.php」「test3.php」はアスキー転送モードでアップロードしてください。

動作

「index.html」をブラウザで開きますと、キャンバスの下にボタンが12個表示されます。

ボタンを押すと、キャンバス左上にリクエスト内容とレスポンス内容が表示され、通信に成功するとキャンバスに画像が表示されます。

クロスドメイン通信

「res」フォルダ、「test1.php」、「test2.php」、「test3.php」を別のサーバーにアップロードし、クロスドメイン通信のテストも行うことができます。

「index.html」の中の変数 SERVER に、通信先サーバーのURLを指定してください。
また、ボタン用の <img> 要素の src 属性も書き換えてください。
