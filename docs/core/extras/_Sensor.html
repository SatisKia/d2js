_Sensor.js

センサーを使用します。

canUseDeviceMotion

canUseDeviceMotion()

デバイスの加速度を取得する機能が使用できるかどうかチェックします。

戻り値：
デバイスの加速度を取得する機能が使用できる場合 true、できない場合 false

canUseDeviceOrientation

canUseDeviceOrientation()

デバイスの向きを取得する機能が使用できるかどうかチェックします。

戻り値：
デバイスの向きを取得する機能が使用できる場合 true、できない場合 false

sensorStart

sensorStart()

センサーの受信を開始します。

sensorStop

sensorStop()

センサーの受信を中断します。

getAccelX

getAccelX()

X軸の移動加速度を取得します。

getGravityZ

getGravityZ()

戻り値：

getLinearAccelY
