_Geolocation.js

端末の位置情報にアクセスします。

canUseGeolocation

canUseGeolocation()

端末の位置情報へのアクセスが有効かどうかチェックします。

戻り値：
位置情報へアクセスできる場合 true、できない場合 false

onGeolocation

onGeolocation(code,message)

端末の位置が変化する度に呼び出されるコールバック関数です。

パラメータ：
code次の値が渡されます。
_GEOLOCATION_ERROR
_GEOLOCATION_PERMISSION_DENIED
_GEOLOCATION_POSITION_UNAVAILABLE
_GEOLOCATION_TIMEOUT
_GEOLOCATION_SUCCESS
message成功時は空文字列、エラー時はエラー内容を表す文字列が渡されます。

コンストラクタ

_Geolocation

_Geolocation()

_Geolocation コンストラクタです。

メソッド

setEnableHighAccuracy

setEnableHighAccuracy(enableHighAccuracy)

可能な限り正確な結果を受け取ることを望むかどうかを指定します。

パラメータ：
enableHighAccuracy可能な限り正確な結果を受け取ることを望む場合 true、望まない場合 false

setMaximumAgeSeconds

setMaximumAgeSeconds(maximumAge)

キャッシュされた位置を返すことが可能な最大時間を秒単位で指定します。

パラメータ：
maximumAgeキャッシュされた位置を返すことが可能な最大時間（秒）

setTimeoutSeconds

setTimeoutSeconds(timeout)

端末が位置取得に費やすことができる最大時間を秒単位で指定します。

パラメータ：
timeout端末が位置取得に費やすことができる最大時間（秒）

start

start()

端末の位置の変化を監視開始します。

stop

stop()

端末の位置の変化を監視停止します。

latitude

latitude()

緯度を取得します。

戻り値：
緯度を 10 進数の角度で返します。

longitude

longitude()

経度を取得します。

戻り値：
経度を 10 進数の角度で返します。

accuracy

accuracy()

取得された緯度および経度の精度を取得します。

戻り値：
取得された緯度および経度の精度をメートル単位で返します。

altitude

altitude()

海面からの相対的な高度を取得します。

戻り値：
海面からの相対的な高度をメートル単位で返します。

altitudeAccuracy

altitudeAccuracy()

取得された高度の精度を取得します。

戻り値：
取得された高度の精度をメートル単位で返します。

heading

heading()

端末の向きを取得します。

戻り値：
端末の向きが真北からどれだけ離れているかを、角度で返します。

補足説明：
0 で真北、180 で真南を表します。方向は時計回りに定められます。

speed

speed()

端末の移動速度を取得します。

戻り値：
端末の移動速度をメートル毎秒で返します。

timestamp

timestamp()

位置情報が取得された時刻を表すタイムスタンプを取得します。

戻り値：
位置情報が取得された時刻を表すタイムスタンプををミリ秒単位の Unix 時刻で返します。
