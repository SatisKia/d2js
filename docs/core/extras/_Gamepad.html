_Gamepad.js

ゲームパッド機能を提供します。

canUseGamepad

canUseGamepad()

ゲームパッド機能が使用できるかどうかチェックします。

戻り値：
ゲームパッド機能が使用できる場合 true、できない場合 false

補足説明：
ブラウザがゲームパッド機能を有しているかのチェックであり、ゲームパッドを認識するかのチェックではありません。ゲームパッドの認識は fetch() で行います。

processGamepadEvent

processGamepadEvent(type,id,param)

各種イベントが通知されたときに呼ばれる関数です。

パラメータ：
typeイベントのタイプが渡されます。
id
paramイベントのパラメータが渡されます。パラメータの意味はイベントによって異なります。

補足説明：
イベントのタイプには、次のものがあります。
_GAMEPAD_BUTTON_PRESSED_EVENT
_GAMEPAD_BUTTON_RELEASED_EVENT
_GAMEPAD_AXIS_MINUS_INPUTTED_EVENT
_GAMEPAD_AXIS_MINUS_RELEASED_EVENT
_GAMEPAD_AXIS_PLUS_INPUTTED_EVENT
_GAMEPAD_AXIS_PLUS_RELEASED_EVENT

コンストラクタ

_Gamepad

_Gamepad()

_Gamepad コンストラクタです。

メソッド

setTolerance

setTolerance(min,max)

パラメータ：
min0.0〜1.0 の値を指定します。
max0.0〜1.0 の値を指定します。

fetch

fetch(index)

指定したインデックスのゲームパッドの接続を確認します。

パラメータ：
indexゲームパッドのインデックスを指定します。

戻り値：
ゲームパッドが認識できない場合、null を返します。

id

id()

コントローラーに関する識別情報を取得します。

戻り値：
コントローラーに関する識別情報を返します。

補足説明：
fetch() で認識されたゲームパッドが対象です。

axisNum

axisNum()

軸の数を取得します。

戻り値：
軸の数を返します。

補足説明：
fetch() で認識されたゲームパッドが対象です。

axisValue

axisValue(axisIndex)

軸の浮動小数点数値を取得します。

パラメータ：
axisIndex軸のインデックスを指定します。

戻り値：
-1.0〜1.0 の値を返します。

補足説明：
fetch() で認識されたゲームパッドが対象です。

buttonNum

buttonNum()

ボタンの数を取得します。

戻り値：
ボタンの数を返します。

補足説明：
fetch() で認識されたゲームパッドが対象です。

isButtonPressed

isButtonPressed(buttonIndex)

ボタンが押されているかどうかチェックします。

パラメータ：
buttonIndexボタンのインデックスを指定します。

戻り値：
ボタンが押されている場合 true、押されていない場合 false を返します。

補足説明：
fetch() で認識されたゲームパッドが対象です。

isButtonTouched

isButtonTouched(buttonIndex)

ボタンがタッチされているかどうかチェックします。

パラメータ：
buttonIndexボタンのインデックスを指定します。

戻り値：
ボタンがタッチされている場合 true、タッチされていない場合 false を返します。

補足説明：
fetch() で認識されたゲームパッドが対象です。

buttonValue

buttonValue(buttonIndex)

ボタンの浮動小数点数値を取得します。

パラメータ：
buttonIndexボタンのインデックスを指定します。

戻り値：
アナログボタンの場合は 0.0〜1.0 の値を返します。アナログボタンでない場合は 0.0 または 1.0 の値が返されます。

補足説明：
fetch() で認識されたゲームパッドが対象です。
