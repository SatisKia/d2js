_Image.js

イメージリソースを読み込む機能、オフスクリーンイメージ機能を提供します。

init() 関数内で以下の設定を有効にすることで、機能を付加できます。
_USE_LOCKIMAGEオフスクリーンイメージ機能を有効にします。

loadImage

loadImage(src)

Image オブジェクトを構築します。

パラメータ：
srcイメージリソースURL

戻り値：
構築された Image オブジェクトを返します。

isImageBusy

isImageBusy()

Image オブジェクトがイメージリソースを読み込み中かどうかを示す値を取得します。

戻り値：
イメージリソースを読み込んでいてビジーな場合は true、それ以外の場合は false を返します。

_Image オブジェクトのコンストラクタ

_Image

_Image(width,height)

_Image コンストラクタです。

パラメータ：
widthオフスクリーンの幅を指定します。
heightオフスクリーンの高さを指定します。

補足説明：
_USE_LOCKIMAGE が有効にされている場合にのみ使用できます。

_Image オブジェクトのメソッド

lock

lock()

オフスクリーンへの描画の開始を宣言します。オフスクリーンはクリアされます。

補足説明：
_USE_LOCKIMAGE が有効にされている場合にのみ使用できます。

unlock

unlock()

オフスクリーンへの描画の終了を宣言します。

補足説明：
_USE_LOCKIMAGE が有効にされている場合にのみ使用できます。

getWidth

getWidth()

オフスクリーンイメージの幅を取得します。

戻り値：
オフスクリーンイメージの幅を返します。

補足説明：
_USE_LOCKIMAGE が有効にされている場合にのみ使用できます。

getHeight

getHeight()

オフスクリーンイメージの高さを取得します。

戻り値：
オフスクリーンイメージの高さを返します。

補足説明：
_USE_LOCKIMAGE が有効にされている場合にのみ使用できます。

getImage

getImage()

オフスクリーンイメージの Image オブジェクトを取得します。

戻り値：
Image オブジェクトを返します。

補足説明：
_USE_LOCKIMAGE が有効にされている場合にのみ使用できます。
