_HttpRequest.js

サーバーと通信する機能を提供します。

httpGet

httpGet(url)

GETリクエストを送信します。

パラメータ：
url通信先URLを指定します。

戻り値：
作成された通信オブジェクトを返します。作成に失敗した場合、null を返します。

補足説明：
通信オブジェクトの作成に成功した場合、以下の関数が呼び出されます。
onHttpResponse(request,data)
通信成功したときに呼び出されます。
request に呼び出し元の通信オブジェクト、data にレスポンスデータが渡されます。
onHttpError(request,status)
通信失敗したときに呼び出されます。
request に呼び出し元の通信オブジェクト、status にHTTPステータスコードが渡されます。

httpPost

httpPost(url,data,type)

POSTリクエストを送信します。

パラメータ：
url通信先URLを指定します。
data送信するデータを指定します。
type送信するデータのMIMEタイプを指定します。

戻り値：
作成された通信オブジェクトを返します。作成に失敗した場合、null を返します。

補足説明：
通信オブジェクトの作成に成功した場合、以下の関数が呼び出されます。
onHttpResponse(request,data)
通信成功したときに呼び出されます。
request に呼び出し元の通信オブジェクト、data にレスポンスデータが渡されます。
onHttpError(request,status)
通信失敗したときに呼び出されます。
request に呼び出し元の通信オブジェクト、status にHTTPステータスコードが渡されます。
