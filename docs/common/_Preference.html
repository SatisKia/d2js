_Preference.js

プリファレンスの読み書きの機能を提供します。
ブラウザが HTML5 ストレージ機能を有し、HTML5 ストレージを使用するように指定した場合、HTML5 ストレージの読み書きが行われます。HTML5 ストレージが無効の場合、Cookie の読み書きが行われます。

コンストラクタ

_Preference

_Preference(useStorage)

_Preference コンストラクタです。

パラメータ：
useStorageHTML5 ストレージを使用する場合 true、使用しない場合 false を指定します。

メソッド

num

num()

保存されているキーの数を取得します。

戻り値：
保存されているキーの数を返します。

getKey

getKey(index)

指定したインデックスのキー名を取得します。

パラメータ：
indexnum() で取得できる数のキーのインデックスを 0 オリジンで指定します。

戻り値：
指定したインデックスのキー名を返します。指定したインデックスのデータが存在しない場合、空文字列を返します。

get

get(key,defValue)

キーに対応する値を取得します。

パラメータ：
keyキー文字列を指定します。
defValueそのキーの値が存在しないときの初期値を指定します。

戻り値：
キーに対応する値を返します。

set

set(key,value)

キー名と値の組み合わせでデータを保存します。

パラメータ：
keyキー文字列を指定します。
valueキーに対応する値を指定します。

clear

clear()
clear(prefix)

データを消去します。

パラメータ：
prefix指定された文字列で始まるキー名のデータのみが対象となります。

beginRead

beginRead(key)

データの読み出しを開始します。

パラメータ：
keyendWrite() により保存されたデータのキー文字列を指定します。

read

read()

データを1個読み出します。

戻り値：
読み出したデータを返します。

補足説明：
本処理呼び出しは beginRead() から endRead() の中で有効です。

endRead

endRead()

データ読み出しを終了します。

beginWrite

beginWrite()

データの書き出しを開始します。

write

write(str)

データを1個書き出します。

パラメータ：
str書き出すデータを指定します。

補足説明：
本処理呼び出しは beginWrite() から endWrite() の中で有効です。

endWrite

endWrite(key)

データ書き出しを終了します。

パラメータ：
keyキー文字列を指定します。
