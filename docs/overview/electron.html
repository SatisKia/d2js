Electronの紹介

Electronを使用しますと、ブラウザアプリをデスクトップアプリにすることができます。

Node.jsのインストール

●ダウンロードサイト
https://nodejs.org/

Electronのインストール

「Node.js command prompt」を起動し、下記のコマンドを実行します。

npm install electron -g
npm i electron-packager -g
npm install asar -g

プロジェクトの初期設定

プロジェクトフォルダ内に移動して、下記のコマンドを実行します。

npm init -y

パッケージ化

Electronのバージョンを確認しておきます。

electron -v

プロジェクトフォルダ内に移動して、下記のコマンドでパッケージ化します。

electron-packager . <アプリ名> --electron-version=<Electronのバージョン> --platform=win32 --arch=x64 --app-copyright=<著作権表記> --app-version=<アプリのバージョン> --icon=<アイコンファイル名> --overwrite

「アプリ名」に指定した名称がフォルダ名に使用され、実行ファイル名になります。
「Electronのバージョン」には、先ほど確認したバージョンを指定します。

x86バイナリを作成する場合は「--arch=x64」を「--arch=ia32」に変えます。
asarにかける場合は「--asar」を付けます。
