開発ツールの準備

D2JSを使って「HTML5 Canvas + JavaScript」アプリを作るために必要な開発ツールは、以下の2つになります。

・UTF-8対応テキストエディタ
・UTF-8対応Cプリプロセッサ

また、次のツールもあると便利です。

・Closure Compiler
・Microsoft Ajax Minifier

HTML5 Canvas をサポートしていない Internet Explorer 7 と 8 向けの場合、次のライブラリが必要です。

・ExplorerCanvas

UTF-8対応テキストエディタ

UTF-8対応テキストエディタなら何でも構いません。

UTF-8対応Cプリプロセッサ

UTF-8対応Cプリプロセッサが必要です。「MinGW」を紹介します。

●MinGW
GNUツールチェーンのWindows移植版です。GNU Cコンパイラを含んでいます。

●ダウンロードサイト
http://www.mingw.org/

Closure Compiler、Microsoft Ajax Minifier

JavaScriptコードの圧縮・難読化ツールです。

●Closure Compiler
http://code.google.com/p/closure-compiler/

●Microsoft Ajax Minifier
http://ajaxmin.codeplex.com/

ExplorerCanvas

Internet Explorer 7 と 8 がネイティブにサポートしていない HTML5 Canvas をJavaScriptレベルで実装するものです。

●ダウンロードサイト
http://code.google.com/p/canvas-text/
